Introduction

Mobile App is about cement ordering by customer via Mobile App provided by EPCCo. This process is meant to facilitate customer to create an order from a mobile using an App and carry the Barcode/QR code or printed order for further activities of sales booking, cement loading and delivery note printing.

Mobile App is collaborated with E-Business Suite Applications of Eastern Cement Province Company's Oracle ERP system.

Mobile App is also meant to collaborate Plant Entry & Exit system of trucks that arrive for cement loading.

Currently the App is available for android mobile users.

Benefits:
  • Replace Print formats with Electronic/Digital formats 
  • Customers can self-order
  • Customer can view electronic Delivery Note  & Invoice on his mobile
  • Customer's  engaged time per order is reduced
